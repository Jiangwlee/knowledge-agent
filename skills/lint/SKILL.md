# Lint

Wiki health check and consistency verification.

## Checks

- Cross-document contradiction detection
- Missing data / gap identification
- Cross-domain connection discovery
- Orphan article detection (no inbound links)
- Index consistency verification (_index/ vs actual files)
- New concept/map candidate suggestions
